# Did You Miss Me Podcast System

## PURPOSE
This folder powers a Netflix-style podcast experience.

## HOW TO ADD EPISODES
1. Open data/episodes.json
2. Duplicate an existing episode object
3. Fill in:
   - title
   - youtubeId
   - thumbnail
   - description
   - blog content

## RULES
- Every episode MUST have a blog section
- Blog should be 600-1200 words for SEO
- Titles should include search keywords

## GOAL
Turn every episode into:
- A video
- A searchable article
- A traffic funnel
